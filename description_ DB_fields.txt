Таблица characters:
id (INTEGER): уникальный идентификатор персонажа (первичный ключ).
name (TEXT): имя персонажа.
role (TEXT): роль персонажа.
description (TEXT): описание персонажа.
photo (BLOB): фотография персонажа.
Порядок взаимодействия: Добавление, обновление и удаление персонажей; получение информации о персонажах для отображения в игре.
Таблица scenes:
id (INTEGER): уникальный идентификатор сцены (первичный ключ).
description (TEXT): описание сцены.
options (TEXT): варианты действий в сцене.
photo (BLOB): фотография сцены.
scene_text (TEXT): текст сцены.
transition_hint (TEXT): подсказка для перехода к следующей сцене.
Порядок взаимодействия: Отображение сцены игроку, получение вариантов действий, сохранение информации о текущей сцене.
Таблица dialogs:
id (INTEGER): уникальный идентификатор диалога (первичный ключ).
scene_id (INTEGER): идентификатор сцены, к которой относится диалог (внешний ключ).
text (TEXT): текст диалога.
changeable_text_id (INTEGER): идентификатор текста, который может изменяться в зависимости от игровых событий (внешний ключ).
Порядок взаимодействия: Отображение диалогов в сцене, изменение текста диалогов в зависимости от игровых событий.
Таблица texts:
id (INTEGER): уникальный идентификатор текста (первичный ключ).
content (TEXT): содержание текста.
variant (TEXT): вариант текста.
Порядок взаимодействия: Получение текстов для отображения в диалогах и сценах.
Таблица relationships:
id (INTEGER): уникальный идентификатор связи (первичный ключ).
from_scene_id (INTEGER): идентификатор исходной сцены (внешний ключ).
to_scene_id (INTEGER): идентификатор целевой сцены (внешний ключ).
condition (TEXT): условие для перехода между сценами.
Порядок взаимодействия: Определение переходов между сценами в зависимости от условий.
Таблица game_state:
id (INTEGER): уникальный идентификатор состояния игры (первичный ключ).
current_scene (INTEGER): идентификатор текущей сцены (внешний ключ).
inventory (TEXT): инвентарь игрока.
status (TEXT): статус игры.
Порядок взаимодействия: Сохранение текущего состояния игры, отображение текущей сцены и инвентаря игрока, проверка статуса игры.
